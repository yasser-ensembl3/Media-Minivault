# Media Vault Documentation

Welcome to Media Vault - your daily entry point for content consumption.

## What is Media Vault?

Media Vault is a read-only frontend that displays content from a single Notion database. It helps you answer: **"What should I read/watch next?"**

## Quick Links

- [Setup Guide](/docs/setup) - Get started with Media Vault
- [Notion Schema](/docs/notion-schema) - Configure your Notion database
- [API Reference](/docs/api-reference) - API endpoints documentation

## Features

| Feature | Description |
|---------|-------------|
| **Filtering** | Filter by type, status, or source |
| **Search** | Quick search by title |
| **Preview** | Embedded preview for Google Docs/Sheets |
| **Add Content** | Quick add form for new items |
| **Keyboard Shortcuts** | Press `R` to refresh |
