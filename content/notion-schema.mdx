# Notion Database Schema

Your Notion database should have the following properties for ContentVault to work correctly.

## Required Properties

| Property | Type | Description |
|----------|------|-------------|
| **Title** (or Name) | Title | The content title |
| **URL** | URL | Direct link to the content |
| **Type** | Select | Content category |
| **Status** | Select | Reading progress |
| **Date Added** | Date | When the content was added |

## Optional Properties

| Property | Type | Description |
|----------|------|-------------|
| Source | Select | Where the content came from |
| Tags | Multi-select | Custom tags for organization |
| Priority | Select | Priority level |
| Notes | Text | Personal notes about the content |

## Suggested Select Options
